Projects

Association for Computing Machinery (ACM) Authentication System Consolidation

Currently, ACM's website (codename: liquid) for its many different functions requires multiple passwords--the University AD password, Korboros password, and LDAP password. This is not only confusing, but also rather ineffective. As part of ACM's admin team, I am working to streamline this mess by unifying all of liquid under the AD password. I am creating a django plug-in that interacts with the Crowd authentication server.

PairTunes

Project I hacked together with two friends for IEEE Hackathon 2013. Using RTP and sockets, we created a server-client Java application that streams synchronized music between computers, meanwhile also syncing libraries and queues. We used the Java Media Framework (JMF) as well as javax.

PyTunes

For the ACM Hackathon, we ported PairTunes to python. We used UofI's NTP (Network Time Protocol) server to overcome issues with ping we encountered while developing PairTunes. I also fiddled around with kytten, a set pyglet libraries, in order to develop a working GUI.

Guitar

A few years ago, I built a guitar out of scratch for the Science Olympiad event Sounds of Music. I had to cultivate my creativity and innovative prowess, as well as learn the true meaning of patience. For example, I had to make many design compromises because home-made tuning pegs cannot withstand the immense tension of guitar strings. I ended up fortifying and moving the pegs to the bottom of the guitar, creating a headless stock guitar. We won first place in state for the event, which was very rewarding for the dozens of hours I had invested in the project.
